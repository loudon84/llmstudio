# 大模型学习与研究项目

## 项目简介

本项目主要用于记录大模型相关的产品学习与技术框架研究。

## 目录结构

```
workspace/
├── README.md                              # 项目说明文档
├── context/                               # 上下文文件目录
├── prompt/                                # 提示词工程目录
├── project/                               # 产品研究目录
└── 使用 Anthropic 构建丰富上下文的 AI 应用程序/  # Anthropic 课程学习笔记
```

## 目录说明

### context 目录

用于保存每次问题需要加载到大模型的上下文文件，提高大模型回答的专业性。

- 存放领域知识文档
- 存放参考资料和背景信息
- 支持大模型理解特定场景的上下文内容

### prompt 目录

用于记录提示词工程训练。

- 收集和整理优质提示词模板
- 记录提示词优化实验过程
- 积累不同场景下的提示词最佳实践

### project 目录

用于生成产品的需求研究、解决方案和功能执行计划。

- 产品需求分析文档
- 技术解决方案设计
- 功能开发执行计划
- 项目迭代规划

## 学习资源

### 使用 Anthropic 构建丰富上下文的 AI 应用程序

本目录包含 Anthropic 官方课程的学习笔记，涵盖以下内容：

- `L3.ipynb` - 课程第三章
- `L4.ipynb` - 课程第四章
- `L5.ipynb` - 课程第五章
- `L6.ipynb` - 课程第六章
- `L7.ipynb` - 课程第七章
- `L9.ipynb` - 课程第九章
- `Appendix.ipynb` - 附录内容

## 使用指南

1. 在 `context` 目录中添加相关领域的上下文文档
2. 在 `prompt` 目录中记录和优化提示词
3. 在 `project` 目录中规划产品需求和解决方案
4. 参考学习资源目录中的笔记进行实践
