## 目录详细说明

### 根目录文件

| 文件 | 说明 |
|------|------|
| `README.md` | 项目主说明文档，包含项目简介、使用指南等 |
| `sitemap.md` | 目录索引文件，记录项目结构与文件说明 |

---

### context/ - 上下文文件目录

**用途**：保存每次问题需要加载到大模型的上下文文件，提高大模型回答的专业性。

**存放内容**：
- 领域知识文档
- 参考资料和背景信息
- 特定场景的上下文内容

| 文件 | 说明 |
|------|------|
| `.gitkeep` | Git 占位文件 |

---

### prompt/ - 提示词工程目录

**用途**：记录提示词工程训练。

**存放内容**：
- 优质提示词模板
- 提示词优化实验记录
- 不同场景下的提示词最佳实践

| 文件 | 说明 |
|------|------|
| `.gitkeep` | Git 占位文件 |

---

### project/ - 产品研究目录

**用途**：生成产品的需求研究、解决方案和功能执行计划。

**存放内容**：
- 产品需求分析文档
- 技术解决方案设计
- 功能开发执行计划
- 项目迭代规划

| 文件 | 说明 |
|------|------|
| `.gitkeep` | Git 占位文件 |
